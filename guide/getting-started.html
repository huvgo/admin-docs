准备工作 #

你需要提前Admin-MS的环境。这是必要的

如果你要开发定制的功能，需要你尽可能的熟悉Spring Boot、Maven、Vue等主要的技术

运行环境 #

  • JDK（1.8及以上版本）
  • Mysql数据库（5.7及以上版本）
  • Redis数据库

开发工具 #

  • 后端
    • IDE
    • Maven
  • 前端
    • Node.js
